# 社区行为准则

Biga 是一个面向投资研究和投资者教育的 Markdown Skill 集合。为了让讨论保持专业、理性和可复用，参与 issue、PR 或其他协作时，请遵守以下准则。

## 基本原则

- 尊重不同观点，围绕事实、证据和分析框架讨论问题。
- 明确区分事实、假设、观点和推测。
- 对不确定性、风险和策略失效条件保持充分披露。
- 使用清晰、克制、可验证的表达，避免情绪化争论。

## 不接受的行为

- 人身攻击、羞辱、骚扰、歧视或恶意挑衅。
- 发布具体荐股、喊单、收益承诺或诱导性交易建议。
- 传播未经核验的数据、传闻、内幕信息或误导性结论。
- 鼓励满仓、杠杆、无止损等忽视风险控制的表达。
- 借项目讨论进行广告、导流、付费荐股或其他商业诱导。

## 投资讨论边界

本项目只讨论投资研究框架、核验清单、策略适用边界和风险提示。涉及个股、行业或主题时，应优先说明分析路径和风险因素，不应给出确定性买卖结论。

## 问题处理

如果你发现不符合本准则的内容，可以通过 GitHub issue 或 PR review 说明问题位置、原因和建议处理方式。维护者可根据具体情况要求修改、关闭讨论或拒绝合并相关内容。
