# Biga 投资研究策略 Skill 集合

Biga 是一个按策略维度组织的投资研究 Skill 文档集合。每个策略目录都是一个独立 Skill 包，包含自己的 `SKILL.md`，用于在对应投资场景下输出研究框架、核验清单、风险提示和分析模板。

## 目录结构

```text
biga/
├── SKILL.md
├── biga-value-skill/
│   └── SKILL.md
├── biga-growth-skill/
│   └── SKILL.md
├── biga-trend-skill/
│   └── SKILL.md
├── biga-dividend-skill/
│   └── SKILL.md
├── biga-cyclical-skill/
│   └── SKILL.md
└── biga-event-skill/
    └── SKILL.md
```

## 使用原则

使用本目录时，先判断用户任务属于哪类投资研究场景，再进入对应策略 Skill：

| 用户需求 | 优先查看 |
|---|---|
| A 股价值投资、公司基本面、ROE/ROIC、现金流、分红、估值、安全边际、价值陷阱分析 | `biga-value-skill/SKILL.md` |
| 成长投资、成长股、行业 ETF、AI/半导体/新能源/创新药、军工、高端制造、软件、消费升级等赛道分析 | `biga-growth-skill/SKILL.md` |
| 趋势投资、趋势交易、右侧交易、均线、成交量、突破、回踩、相对强弱、止损线分析 | `biga-trend-skill/SKILL.md` |
| 股息策略、红利、高股息、稳定分红、收息、红利 ETF、银行/运营商/电力/煤炭/高速公路/公用事业分红分析 | `biga-dividend-skill/SKILL.md` |
| 周期投资、周期股、资源股、商品周期、煤炭、有色、钢铁、化工、航运、石油、建材、农业周期分析 | `biga-cyclical-skill/SKILL.md` |

## 策略 Skill

当用户讨论 A 股价值投资、公司基本面、财务质量、盈利质量、现金流、分红、估值、安全边际、价值陷阱等主题时，使用：

```text
biga-value-skill/SKILL.md
```

当用户讨论成长投资、成长股、行业 ETF、AI、半导体、新能源、创新药、军工、高端制造、软件、消费升级等主题时，使用：

```text
biga-growth-skill/SKILL.md
```

biga-trend-skill/SKILL.md

## 投资内容边界

- 输出定位为投资研究和投资者教育，不构成直接投资建议。
- 不承诺收益，不输出“稳赚”“必涨”“满仓”等结论。
- 涉及个股时，应优先提供分析框架、核验清单和风险提示。
- 涉及交易纪律时，应强调仓位控制、止损规则和情景复盘。
- 涉及高波动主题或周期资产时，应提示估值、拥挤度、业绩兑现和回撤风险。
